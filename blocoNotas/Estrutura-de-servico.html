Modelo

Modelo seria a cópia estrutura do banco de dados.
Iremos criar um molde, referente ao nosso banco de dados.

Vai na árvore de componentes do lado esquerdo,
crie um novo "arquivo" dentro da pasta "curso" com o nome curso.ts

É atraves desse arquivo que teremos acesso a nossa tabela,banco de dados.

Dentro desse arquivo, coloque :

export class curso () *abre chaves*

constructor (*){} => *aqui passa os parâmetros para efetuar alguma ação.

// Veja como deve ficar:
__constructor(

________nomeCurso: string,
________valorCurso: number,
________idCurso: number

__){}

*fecha a chave*


Veja como é o código:


É importante colocar o nome e o valor por que provavelmente será usado,
já o id, ás vezes precisa e as vezes não, por isso coloca a interrogação ?.


Sempre que for trabalhar com banco de dados precisa ter essa estrutura,
do banco de dados, e um constructor, para manipular as informações,
que serão enviadas e recebidas pelo banco de dados.

Serviços

Serve como um intermediário, para enviar para as camadas de serviço.

Para criar esse serviço, vai no pront de comando do "VScode",
certifique que você está dentro da pasta do programa, coloque:

ng g s curso/curso dê enter.



Explicando o código:

ng => comando do Angular.
g => generate , gerar algo.
s => service, diz que será um serviço.
curso/ => pasta onde vai ficar o componente criado.
curso => nome do serviço.
