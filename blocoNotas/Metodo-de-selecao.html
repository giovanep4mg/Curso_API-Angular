url = "http://localhost/api/php/";

Agora vamos colocar no ngOnInit( ){ } ,

// ao iniciar o sistema, deverá listar os cursos no front-end "página"

Exibir os cursos no front-end (página)

Vamos exibir os cursos no front-end.

No componente curso.component.ts adicione um objeto.

Vamos criar um objeto, da classe "curso" para poder instânciar,
Para pode usar todos os dados que estão lá: id, nomeCurso, valorCurso.

// objeto da classe curso
curso = new Curso( );


Para exibir, vai no componente curso.component.html,
pode apagar os botões que estão lá.

Agora adicione essas tags:

table border = "1" => cria uma tabela, com borda.
// vai percorre toda tabela

__tr *ngFor="let v of vetor" => linha da tabela
_____td { { v.nomeCurso } } /td => coluna da tabela.
_____td { { v.valorCurso } } /td => coluna da tabela.
__tr => fecha a linha da tabela.
table => fecha a tabela
