Configurando o Angular

É importante já ter instalado no seu computador o node.js,
para conseguir baixar a dependência do Angular.
Baixar também o Angular .

Abra o "VScode", no arquivo, escolha abri pasta, procure a pasta "api",
que está dentro da pasta "xampp", na pasta "htdocs".

Abra o terminal do angular, verifique se você está na pasta "api",

Para entra numa pasta digite: cd *nome da pasta* .
Para sair da pasta digite: cd...

coloque esse código ng new app-php, vai criar no projeto angular.

Vai te dá algumas opções de escolha, coloque "y" e escolha o "CSS".
Clique em enter, vai começar a criar o projeto, pode demorar um pouquinho,

Depois que criar o projeto, volte novamente no terminal do "VScode",
verifique se está na pasta do projeto criado:
C:\xampp\htdocs\api\app-php

Para entra numa pasta digite: cd *nome da pasta* .
Para sair da pasta digite: cd...

Digite esse comando: ng serve --open ,
na primeira vez que for colocado esse código vai pedir para confirma,
se você aceita os termos ou não, coloque "y".

Vai abrir o projeto no navegador do seu computador.
Se não abrir, volta e faça tudo novamente por que teve algum erro.


Adicionando Modulos

Na pasta que foi criado o projeto, que está do lado esquerdo,
clique em src e dentro dessa pasta procure o arquivo app.module.ts

Dentro dele terá algumas configurações do angular,
o que queremos no nosso projeto é enviar e receber dados.

Para isso precisa fazer duas importações importantes:

import { HttpClientModule} from '@angular/common/http'

import { FormsModule } from '@angular/forms';

Agora no "ngModule", dentro do imports adicione:

HttpClientModule, => para fazer o vincúlo entre o PHP e o Banco de Dados.

FormsModule => para trabalhar com os dados recebidos e enviados.

Criando Componente

Componentes são partes do nosso sistema,
um segmento, uma parte do site.

No pront de comando do "VScode", verifique se está dentro da pasta do projeto.

Para entra numa pasta digite: cd *nome da pasta* .
Para sair da pasta digite: cd...

coloque esse código:

ng g c curso clique em enter

ng => significa "Angular" .
g => significa "generate" .
c => significa "component" .
curso => significa "nome que você quiser dá " .

Vai criar uma pasta dentro da pasta app, e dentro dela terá:

curso.component.html => trabalhar com o "HTML".
curso.component.css => trabalhar com o "CSS".
curso.component.spect.ts => trabalhar para exportar o "TypeScript".
curso.component.ts => trabalhar com o "TypeScript".
